# 原题
求在1到n个数中挑选k个数的所有的组合类型。

注意点：

  - 每个数字只能够用一遍
  - 组合的排列没有顺序要求

例子：

输入: n = 4, k = 2

输出: [[1, 4], [2, 4], [3, 4], [1, 3], [2, 3], [1, 2]]

# 解题思路
### 回溯法
通过深度优先搜索，回溯出所有可能性即可。

复杂度: 时间 O(N) 空间 O(K)

### 递归
在n个数中选k个，如果n大于k，那么可以分类讨论，
如果选了n，那么就是在1到(n-1)中选(k-1)个，否则就是在1到(n-1)中选k个。
递归终止的条件是k为1，这时候1到n都符合要求。
