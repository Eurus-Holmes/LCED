# 原题
求x的n次幂。

注意点：

  - -100.0 < x < 100.0
  - n is a 32-bit signed integer, within the range [−2^31, 2^31 − 1]

例子：

输入: x = 2, n = -1

输出: 0.5

输入: x = 2.1, n = 2

输出: 4.41

# 解题思路
pow(x,n)就是求x的n次方。

x的n次方可以看做:

$$ x^n = x^{(n/2)} * x^{(n/2)} * x^(n /mod 2) $$

所以利用递归求解，当n==1的时候，x^n=x。

当然n是可以小于0的，2^(-3) = 1/(2^3)。按照上面那个规律就可以解决了。
