# 原题
给一个由包含一串数字的列表组成的非负整数加上一。

注意点：

  - 列表前面的数字表示高位
  - 注意最高位也可能进位

例子：

输入: [1, 2, 3, 4, 9]

输出: [1, 2, 3, 5, 0]

# 解题思路
先对原数组进行处理。从数组最后一位开始往前检查，如果当前数字是<9的，说明你加1无需进位，从循环跳出即可，如果当前数字等于9，
说明加1涉及进位，且加1后当前数字应记为0，继续循环处理。
当对原数组处理完后，还需要判断当前第0位是不是已经变为0了，如果已经变为0了说明是类似99+1这种，需要进位。其他则不需要。
一般对数字进行操作的题都要考虑边界，尤其是溢出问题。
