# 原题
给定一个包含n个正整数的数组x。
从(0, 0)点出发先向北移动x[0]米，然后向西移动x[1]米，再向南移动x[2]米，再向东移动x[3]米，以此类推。
换言之，每一次移动之后都逆时针旋转一次方向。

注意点：

  - 无

例子：

```
Example 1:

┌───┐
│   │
└───┼──>
    │

Input: [2,1,1,2]
Output: true

Example 2:

┌──────┐
│      │
│
│
└────────────>

Input: [1,2,3,4]
Output: false 

Example 3:

┌───┐
│   │
└───┼>

Input: [1,1,1,1]
Output: true 
```

# 解题思路
思路来得蛮快的，因为cross的情况一共就三种，4弯cross，5个弯cross和6个弯cross:

```
#4个弯          5个弯            6个弯

  1              1                1
┌───┐          ┌───┐            ┌───┐
│2  │0         │2  │0           │   │0
└───┼>         │   ↑            │2  │←┐
  3            └───┘5           │   5 │4 
                 4              └─────┘
                                   3
```
根据上图写出三种情况的判断式.
