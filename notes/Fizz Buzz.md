# 原题
编写程序输出数字1-n的字符串形式。

但是对于3的倍数输出 “Fizz”，5的倍数输出“Buzz”。既是3的倍数，又是5的倍数输出“FizzBuzz”。

注意点：

  - 无

例子：

```
n = 15,

Return:
[
    "1",
    "2",
    "Fizz",
    "4",
    "Buzz",
    "Fizz",
    "7",
    "8",
    "Fizz",
    "Buzz",
    "11",
    "Fizz",
    "13",
    "14",
    "FizzBuzz"
]
```
